# Coding Skill

Khi được yêu cầu viết code hoặc sửa code:

1. **Đọc context trước** — Luôn đọc file liên quan trước khi sửa
2. **Giải thích trước khi làm** — Nói rõ sẽ thay đổi gì
3. **Backup quan trọng** — Nếu file quan trọng, đọc trước để confirm
4. **Test sau khi sửa** — Chạy lệnh verify nếu có thể
5. **Không viết code ảo** — Chỉ dùng API/library thật sự tồn tại
